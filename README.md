# CabinSI
This is the official repository for the ECCV 2026 accepted paper 'CabinSI: Omni-Cabin Spatial Reasoning through Explicit Visual Cognitive Maps.' The repository currently includes the json_file folder containing the JSON annotation files, while the corresponding images are undergoing data anonymization. The complete evaluation code and full dataset will be released alongside the official publication of the paper. Thank you for your interest in CabinSI, and you are very welcome to use it!
